```python
import numpy as np

def priority(item: float, bins: np.ndarray) -> np.ndarray:
    """Returns priority with which we want to add item to each bin.

    Args:
        item: Size of item to be added to the bin.
        bins: Array of capacities for each bin.

    Return:
        Array of same size as bins with priority score of each bin.
    """
    return np.where(bins - item > 0, 0, -bins + item)
```

**Explanation:**

The new function uses the `np.where` function to directly assign a positive priority score to bins that are not over capacity (i.e., `bins - item <= 0`), and a negative priority score to bins that are over capacity (i.e., `bins - item > 0`). This approach eliminates the need for the negative sign in the original function and improves performance.
