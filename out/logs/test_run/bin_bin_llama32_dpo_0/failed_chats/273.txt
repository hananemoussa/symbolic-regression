```python
def priority(item: float, bins: np.ndarray) -> np.ndarray:
    """Returns priority with which we want to add item to each bin.

    Args:
        item: Size of item to be added to the bin.
        bins: Array of capacities for each bin.

    Return:
        Array of same size as bins with priority score of each bin.
    """
    bin_idx = np.digitize(item, bins)
    # Assign the priority score based on the bin index
    return (bins - bins[bin_idx]) * (1 - bins[bin_idx])
```

**How it Works:**

1.  The `np.digitize()` function is used to assign a bin index to the item. This is done by comparing the item size to each bin capacity and selecting the bin that the item falls into.
2.  The priority score is then calculated as `(bins - bins[bin_idx]) * (1 - bins[bin_idx])`. This formula is based on the scoring system used in the existing functions. It subtracts the bin capacity from the bin that the item falls into, and then multiplies the result by `(1 - bin_idx)`. This effectively penalizes items that are added to bins with lower capacities.
