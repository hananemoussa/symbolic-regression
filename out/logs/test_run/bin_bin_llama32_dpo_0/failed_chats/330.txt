### Efficient Weighted Random Sampling

We will utilize the C-average sampling technique to efficiently sample items to bins. This technique ensures that samples are drawn from all bins proportionally.

```python
import numpy as np

def priority(item: float, bins: np.ndarray) -> np.ndarray:
    """
    Returns priority with which we want to add item to each bin.

    Args:
        item: Size of item to be added to the bin.
        bins: Array of capacities for each bin.

    Return:
        Array of same size as bins with priority score of each bin.
    """
    # Handle edge case: item capacity is 0
    if item == 0:
        return np.full_like(bins, -1)

    # Sort bins by capacity in descending order
    bins = np.sort(bins)[::-1]

    # Initialize weights for weighted random sampling
    weights = bins.copy()

    # Normalize weights
    weights /= weights.sum()

    # C-average sampling technique
    sample_index = np.random.choice(len(bins), p=weights)

    # Get the bin index and weight
    bin_index = sample_index
    weight = weights[bin_index]

    # Assign priority score
    return -weight
```
