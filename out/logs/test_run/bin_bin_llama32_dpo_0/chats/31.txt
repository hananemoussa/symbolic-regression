import numpy as np

def priority(item: float, bins: np.ndarray) -> np.ndarray:
    """
    Returns a priority array where the priority of each bin is determined by its distance from the minimum.

    Args:
        item: Size of item to be added to the bin.
        bins: Array of capacities for each bin.

    Return:
        Array of same size as bins with the priority of each bin.
    """
    # Calculate the minimum value in the bins array
    min_val = np.min(bins)
    
    # Calculate the distance of each bin from the minimum
    distances = np.abs((bins - min_val) / min_val)
    
    # Assign higher priority to bins with lower distances
    return distances * -bins

# Example usage:

In this new function, we first calculate the minimum value in the `bins` array. Then we calculate the absolute difference between each bin and the minimum, normalized by the minimum. We multiply this normalized difference by the capacity of each bin, which gives us the priority of each bin.
