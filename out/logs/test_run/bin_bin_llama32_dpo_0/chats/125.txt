### Priority Function with Novel Approach

After analyzing the characteristics of the higher-scoring function, we identified that it subtracts the difference between the available capacity and the item size from the item size. We decided to modify this function to add the opposite value to the item size, effectively "moving" the available capacity towards the item size. This is based on the observation that if the available capacity is subtracted from the item size, it implies that there is some capacity left over for the item.

```python
import numpy as np

def priority(item: float, bins: np.ndarray) -> np.ndarray:
    """
    Returns priority with which we want to add item to each bin.
    
    Args:
        item: Size of item to be added to the bin.
        bins: Array of capacities for each bin.

    Return:
        Array of same size as bins with priority score of each bin.
    """
    # Calculate the difference between available capacity and item size
    avail_capacity_diff = bins - item
    
    # Add this difference to the item size to get the priority score
    return -avail_capacity_diff
```
