```python
import numpy as np

def priority(item: float, bins: np.ndarray) -> np.ndarray:
    """
    Returns priority with which we want to add item to each bin.

    Args:
        item: Size of item to be added to the bin.
        bins: Array of capacities for each bin.

    Return:
        Array of same size as bins with priority score of each bin.
    """
    # Sort bins in descending order of their capacities
    bins = np.sort(bins)[::-1]

    # Assign priority scores
    priority_scores = np.sum(bins >= item, axis=1)

    return priority_scores

# Example usage
bins = np.array([2, 3, 5, 1, 4])
item_sizes = np.array([1, 2, 3, 4, 5])

priority_scores = priority(item_sizes, bins)

print("Priority Scores:", priority_scores)
```

This new function, `priority()`, first sorts the bins in descending order of their capacities. Then it assigns a priority score to each item size by summing up the positions of the bins where the item size falls within the range of each bin. This is based on the understanding that we want to add the item size to the bin with the largest capacity, as per the greedy algorithm principle.

In this approach, we are essentially using a greedy strategy, where we prioritize adding items to the bin with the largest available capacity. This is more efficient than the existing approaches, which may not always result in the optimal solution.
