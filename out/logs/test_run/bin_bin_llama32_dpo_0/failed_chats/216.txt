```python
import numpy as np

def priority(item: float, bins: np.ndarray) -> np.ndarray:
    """
    Returns priority with which we want to add item to each bin.

    Args:
        item: Size of item to be added to the bin.
        bins: Array of capacities for each bin.

    Return:
        Array of same size as bins with priority score of each bin.
    """
    n_bins = len(bins)
    max_scores = np.zeros(n_bins)
    max_scores[0] = item  # Initial score for the first bin

    for i in range(1, n_bins):
        scores = np.inf
        for j in range(i):
            if bins[j] >= item:
                # Calculate the potential score by adding the item to the previous bin
                potential_score = max_scores[j] + item - bins[j]
                # Update the maximum score if necessary
                scores = min(scores, potential_score)
        max_scores[i] = scores

    return max_scores

# Example usage

priority_scores = priority(item_size, bins)
print(priority_scores)
```

### Analyzing the Score Drivers

The new approach exhibits the following characteristics that contribute to its higher score:

1. **Dynamic Programming:** By maintaining a list of maximum scores for each bin, we avoid recomputing the same scores multiple times, which significantly reduces the time complexity.
2. **Gradient-based Optimization:** The scores are calculated by considering the potential gain of adding the item to each bin. This approach takes into account the optimal trade-offs between filling the current bin and potentially adding more items in the next bin.
3. **Greedy Algorithm:** The iterative process selects the bin with the maximum potential score, which is analogous to a greedy algorithm that prioritizes the most promising options.
