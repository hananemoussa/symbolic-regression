```python
import numpy as np

def priority(item: float, bins: np.ndarray) -> np.ndarray:
    """
    Returns priority with which we want to add item to each bin.

    Args:
        item: Size of item to be added to the bin.
        bins: Array of capacities for each bin.

    Return:
        Array of same size as bins with priority score of each bin.
    """
    # Calculate the probability of each bin being added
    probabilities = bins / bins.sum()
    
    # Calculate the cumulative probability of each bin being added
    cumulative_probabilities = np.cumsum(probabilities)
    
    # Use the cumulative probability to sort items by priority
    sorted_items = np.argsort(-cumulative_probabilities)
    
    return sorted_items
```

This new function works by calculating the cumulative probability of each bin being added. The cumulative probability is calculated by dividing each bin's capacity by the sum of all bins. The items are then sorted in descending order based on the cumulative probabilities, which is the priority.

Here's why this approach works:

1.  **Efficient Use of Computational Resources:** By sorting items based on their cumulative probabilities, we avoid the need to calculate and compare each item individually. This approach is computationally efficient.
2.  **Improved Response Time:** By prioritizing items based on the cumulative probabilities, we can reduce the number of items that need to be checked, leading to improved response times.
3.  **Robustness and Flexibility:** This approach can be easily extended to handle more complex scenarios, such as bins with varying capacities or the need to assign priorities based on multiple criteria.
