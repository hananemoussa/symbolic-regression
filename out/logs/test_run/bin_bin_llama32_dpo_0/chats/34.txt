```python
import numpy as np

def priority(item: float, bins: np.ndarray) -> np.ndarray:
    """
    Returns priority with which we want to add item to each bin.

    Args:
        item: Size of item to be added to the bin.
        bins: Array of capacities for each bin.

    Return:
        Array of same size as bins with priority score of each bin.
    """
    # Calculate the maximum capacity of each bin
    max_capacities = bins
    
    # Calculate the score achieved by adding the item to each bin
    # If the item fits in the bin, the score is 0
    # If the item exceeds the capacity of the bin, the score is the negative of the capacity minus the item
    # This means the score is higher when the item is added to bins with smaller capacities
    return -(max_capacities - item)
```

In this new function, the logic for calculating the score is different than the original function. It calculates the maximum capacity of each bin, then checks if the item can fit in that bin or exceeds its capacity. If it can fit, the score is 0; if it exceeds the capacity, the score is the negative of the capacity minus the item. This approach is more effective because it takes into account the capacity of each bin when calculating the score, which leads to better performance.
