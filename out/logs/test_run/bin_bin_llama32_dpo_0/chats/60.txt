```python
import numpy as np

def priority(item: float, bins: np.ndarray) -> np.ndarray:
    """Calculates the priority score for each item.

    Args:
    item: Size of item to be added to the bin.
    bins: Array of capacities for each bin.

    Return:
    Array of same size as bins with priority score of each bin.
    """
    # Calculate the priority score based on the item size and bin capacity
    # We use the formula: (item_size - bin_capacity) * bin_capacity
    # This ensures that bins with lower capacity scores are considered higher
    scores = (item - bins) * bins
    return scores
```

**Explanation and variations:**

*   The new function uses the formula `(item - bins) * bins` to calculate the priority score. This ensures that bins with lower capacity scores are considered higher.
*   We use the `numpy` library to handle array operations. Specifically, we use vectorized operations to calculate the scores for each bin in a single operation.
