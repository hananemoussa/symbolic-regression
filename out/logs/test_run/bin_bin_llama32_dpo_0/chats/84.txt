```python
import numpy as np

def priority(item: float, bins: np.ndarray) -> np.ndarray:
    """
    Returns priority with which we want to add item to each bin.
    
    Args:
        item: Size of item to be added to the bin.
        bins: Array of capacities for each bin.
    
    Return:
        Array of same size as bins with priority score of each bin.
    """
    
    # Sort bins by capacity in descending order
    bins = np.sort(bins)[::-1]
    
    # Initialize the priority score for each bin
    bin_scores = np.zeros(len(bins))
    
    # Assign priority to each item in descending order of bin capacity
    for i, capacity in enumerate(bins):
        bin_scores[i] = capacity - item
    
    # Return the priority score of each bin
    return bin_scores
```

## Explanation:

*   The existing `priority()` function uses the formula `- bins - item` to calculate the priority score. However, this approach is biased towards items in bins with larger capacities.
*   To address this bias, we sort the bins in descending order of their capacities before assigning priority scores.
*   We then iterate through each bin and calculate its priority score by subtracting the item size from the bin capacity. This ensures that items in bins with larger capacities are assigned lower priority scores.
*   The final priority scores are returned as an array of the same size as the bins.

## Example Usage:

```python
# Create an array of bins with capacities

# Calculate the priority scores
```
