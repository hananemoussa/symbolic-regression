```python
import numpy as np

def priority(item: float, bins: np.ndarray) -> np.ndarray:
    """
    Returns priority with which we want to add item to each bin.

    Args:
        item: Size of item to be added to the bin.
        bins: Array of capacities for each bin.

    Return:
        Array of same size as bins with priority score of each bin.
    """
    # Calculate the ceiling division of the item's size with the bin's capacity
    # This allows the item to fill the bin if it's a whole number
    # and pads the bin with the remaining value if it's not a whole number
    bins[~np.isnan(bins)] = np.ceil(bins[~np.isnan(bins)] / item)
    
    # Calculate the priority score by taking the negative logarithm of the bin's capacity
    # This is based on the idea that the priority score is inversely proportional to the bin's capacity
    return -np.log(bins)
```

### Explanation of Insights and Innovations

*   **Ceiling Division**: We calculate the ceiling division of the item's size with the bin's capacity. This allows the item to fill the bin if it's a whole number, and pads the bin with the remaining value if it's not a whole number.
*   **Negative Logarithm**: We take the negative logarithm of the bin's capacity. This is because the logarithm function is monotonically increasing, which means that a smaller log value corresponds to a larger bin capacity.

### Insights and Variations

*   **Negative Logarithm**: Using the negative logarithm allows for a more accurate representation of the priority score. This is because the logarithm function is well-behaved, and taking the negative logarithm helps to avoid negative values that may occur when the item size is not a whole number.
*   **Ceiling Division**: Using the ceiling division ensures that the item fills the bin completely, which can be important in some applications.
*   **Padded Bin Calculation**: Calculating the bin capacity for items that are not whole numbers can lead to empty bins. Using the ceiling division and negative logarithm helps to ensure that the bin remains non-empty.
