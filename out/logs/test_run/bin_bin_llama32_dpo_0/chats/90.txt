```python
import numpy as np

def priority(item: float, bins: np.ndarray, num_bins: int = 10, max_steps: int = 1000) -> np.ndarray:
    """
    Returns a priority score for each bin, indicating the most efficient bin to add the item.

    Args:
        item: Size of item to be added to the bin.
        bins: Array of capacities for each bin.
        num_bins: Number of bins to consider (default=10).
        max_steps: Maximum number of iterations (default=1000).

    Return:
        Array of same size as bins with priority score of each bin.
    """
    scores = np.zeros_like(bins)
    scores[0] = -(bins[0] - item)  # Initialize with the most efficient bin

    for i in range(1, num_bins):
        # Check if the previous bin was empty
        if i == 1:
            scores[i] = -(bins[i] - item) if bins[i] > item else 0
        else:
            scores[i] = -np.min((bins[:i] - item) / bins[:i])

        # Select the bin with the highest score
        if scores[i] < scores[i-1]:
            scores[i] = scores[i-1]

        # Check if we've reached the maximum number of iterations
        if i == max_steps:
            break

    return scores
```

This function takes into account not only the capacity of the bin but also the proportion of the item in the previous bin. By using a binary search-like approach, it efficiently selects the most efficient bin to add the item.
