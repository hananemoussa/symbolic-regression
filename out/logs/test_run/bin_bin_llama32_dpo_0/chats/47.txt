### Modified Priority Function

The original function `priority(item: float, bins: np.ndarray) -> np.ndarray` was designed to find the smallest remaining capacity in each bin after adding an item. A slight modification can make it more beneficial by utilizing a greedy algorithm that always adds the item to the bin with the most remaining capacity.

```python
import numpy as np

def priority(item: float, bins: np.ndarray) -> np.ndarray:
    """
    Returns priority with which we want to add item to each bin.
    
    Args:
        item: Size of item to be added to the bin.
        bins: Array of capacities for each bin.

    Return:
        Array of same size as bins with priority score of each bin.
    """
    # Sort bins in descending order
    bins = np.sort(bins)[::-1]
    
    # Initialize priority scores
    priority_scores = np.zeros_like(bins)
    
    # Assign item to bin with the most remaining capacity
    i = 0
    while i < len(bins) and item > bins[i]:
        priority_scores[i] = -(bins[i] - item)
        i += 1
    
    return priority_scores
```
