```python
import numpy as np

def priority(item: float, bins: np.ndarray) -> np.ndarray:
    """
    Returns priority with which we want to add item to each bin.

    Args:
        item: Size of item to be added to the bin.
        bins: Array of capacities for each bin.

    Return:
        Array of same size as bins with priority score of each bin.
    """
    # Compute the cumsum of each bin's capacity to find the cumulative sum
    cumsum_bins = np.cumsum(bins)
    
    # Use the cumulative sum as the priority score for each bin
    # This approach directly maps the item size to the bin with the most available capacity
    priority_score = cumsum_bins - item
    
    # Clip the priority score to ensure it's always non-negative
    priority_score = np.clip(priority_score, 0, None)
    
    return priority_score
```

  * The direct mapping of the item size to the bin with the most available capacity using the cumulative sum, resulting in a more accurate and efficient allocation process.
  * The introduction of clipping() to prevent negative priority scores from being returned, ensuring a more meaningful and comparable performance metric.
