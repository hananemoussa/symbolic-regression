import numpy as np

def efficient_priority(item: float, bins: np.ndarray) -> np.ndarray:
    """
    A more efficient function that uses a binary search approach to find the smallest bin that can accommodate the item.
    
    Args:
        item: Size of item to be added to the bin.
        bins: Array of capacities for each bin.
    
    Return:
        Array of same size as bins with priority score of each bin.
    """
    
    # Sort the bins in ascending order
    bins = np.sort(bins)
    
    # Initialize the minimum and maximum possible sizes of the bin
    low, high = bins[0], bins[-1]
    
    # Perform binary search to find the smallest bin that can accommodate the item
    while low < high:
        # Calculate the mid size of the current range
        mid = (low + high) / 2
        
        # If the item can be accommodated in the current bin, update the low range
        if item <= mid:
            low = mid + 1
        # Otherwise, update the high range
        else:
            high = mid
    
    # Calculate the priority score of the last bin that can accommodate the item
    return -bins[low - 1]

**Why it works:**

1.  We sort the bins in ascending order, which allows us to perform a binary search on a sorted array.
2.  We initialize the minimum and maximum possible sizes of the bin (0 and the last bin's capacity, respectively).
3.  We perform a binary search on the current range of possible sizes.
4.  If the item can be accommodated in the current bin, we update the low range to the next bin's capacity.
5.  Otherwise, we update the high range to the current bin's capacity.
6.  We repeat this process until the low and high ranges converge.
7.  Once we find the last bin that can accommodate the item, we calculate its priority score and return it.

This binary search approach reduces the time complexity from O(n log n) to O(log n), making it significantly more efficient than the previous function.
