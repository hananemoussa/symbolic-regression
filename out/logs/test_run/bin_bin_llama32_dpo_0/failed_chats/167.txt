### Improved Priority Function with Novel Approach

The original function `priority(item: float, bins: np.ndarray) -> np.ndarray` attempts to calculate the priority score by inverting the difference between the bin capacities and the item size. However, this approach can be improved upon by leveraging the properties of binary linear programming.

#### Theoretical Background

To implement this improved function, we'll leverage the fact that minimizing the difference between the bin capacities and the item size can be reduced to finding the optimal allocation of items to bins using linear programming.

#### New Implementation

```python
import numpy as np

def priority(item: float, bins: np.ndarray) -> np.ndarray:
    """
    Returns priority with which we want to add item to each bin.

    Args:
        item: Size of item to be added to the bin.
        bins: Array of capacities for each bin.

    Return:
        Array of same size as bins with priority score of each bin.
    """
    # Initialize the coefficient matrix for linear programming
    A = np.vstack((bins[:, None], 1 - bins))

    # Add a column of ones to the coefficient matrix for the boundary condition
    A = np.hstack((A, np.ones((bins.shape[0] + 1, 1))))

    # Use numpy's least squares solver to find the optimal allocation
    return np.linalg.lstsq(A, np.ones(bins.shape[0]), rcond=None)[0]
```

### Explanation and Justification

This new function uses the least squares method to solve the linear programming problem, which is a well-established technique for finding optimal solutions in various optimization problems. The key to the solution lies in the creation of a coefficient matrix `A` that represents the coefficients of the variables (bin capacities and the item size) in the linear programming problem.

By including a column of ones in `A`, we effectively add a non-zero value to all elements except the last one, which represents the boundary condition for the bin capacities. This allows the least squares solver to find the optimal allocation that minimizes the difference between the bin capacities and the item size.

The `np.linalg.lstsq(A, np.ones(bins.shape[0]), rcond=None)[0]` call returns the optimal allocation vector, which represents the priority scores for each bin.
