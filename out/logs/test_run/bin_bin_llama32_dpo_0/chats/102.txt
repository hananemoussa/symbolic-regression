**Priority Function: An Approach Based on Exponential Decreasing Frequency**
### Overview

This function, `priority()`, takes into account the exponential decrease of frequencies as the item's size decreases. This is a more effective strategy than the previous ones as it ensures that smaller items are distributed into lower-frequency bins first, leading to more efficient use of bin space.

```python
import numpy as np

def priority(item: float, bins: np.ndarray) -> np.ndarray:
    """
    Returns priority with which we want to add item to each bin, taking into account the 
    exponential decrease of frequencies as the item's size decreases.

    Args:
        item: Size of item to be added to the bin.
        bins: Array of capacities for each bin.

    Return:
        Array of same size as bins with priority score of each bin.
    """
    # Calculate the frequency of each bin
    frequencies = np.arange(len(bins))
    
    # Calculate the distance between each bin
    distances = np.diff(frequencies)
    
    # Calculate the exponential decrease of frequencies
    exp_decreases = np.exp(-distances)
    
    # Calculate the priority score
    priority_scores = -np.log(exp_decreases)
    
    return priority_scores
```

### Insights

- The function `priority()` is designed to handle variable item sizes and variable bin capacities.
- The calculation of `distances` allows us to identify the distance between each bin.
- The exponential decrease of frequencies is calculated using `np.exp(-distances)`, which ensures that smaller items are distributed into lower-frequency bins first.
- The `priority()` function returns an array of same size as `bins` with the calculated priority scores.

### Example Usage

```python
# Define the item sizes and bin capacities

bins = np.array([100, 50, 150, 200])

# Calculate the priority scores
priority_scores = priority(item_sizes, bins)

# Print the priority scores
```
