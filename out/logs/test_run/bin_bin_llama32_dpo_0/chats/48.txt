```python
import numpy as np

def priority(item: float, bins: np.ndarray) -> np.ndarray:
    """
    Returns priority with which we want to add item to each bin.

    Args:
        item: Size of item to be added to the bin.
        bins: Array of capacities for each bin.

    Return:
        Array of same size as bins with priority score of each bin.
    """
    # Calculate the relative priority score for each bin
    # The score is the bin's capacity minus the item's size relative to the bin's capacity
    relative_scores = (bins - item) / bins
    # Use np.clip to ensure the scores are within the valid range
    relative_scores = np.clip(relative_scores, 0, 1)
    return relative_scores
```

### Explanation
The new function uses a relative priority score calculation that accurately represents the task of determining priority with which we add an item to each bin.

1.  Calculate the relative priority score for each bin by subtracting the item's size from the bin's capacity and dividing the result by the bin's capacity.
2.  Use `np.clip` to ensure the scores are within the valid range of 0 to 1, which corresponds to the probability distribution of bin capacities.
3.  Return the relative priority scores as the output array.
