```python
import numpy as np

def priority(bins, item):
    """
    Returns priority with which we want to add item to each bin.

    Args:
        bins: Array of capacities for each bin.
        item: Size of item to be added to the bin.

    Return:
        Array of same size as bins with priority score of each bin.
    """
    # Sort bins in descending order of capacity
    bins = np.sort(bins)[::-1]

    # Calculate cumulative capacity
    cumulative_capacity = np.cumsum(bins)

    # Initialize priority scores
    scores = np.zeros_like(bins)

    # Calculate priority scores
    for i, capacity in enumerate(cumulative_capacity):
        scores[i] = -((capacity - item) / capacity)

    return scores
```

This new function works as follows:

1.  Sort the bins in descending order of their capacities.
2.  Calculate the cumulative capacity of each bin.
3.  Initialize a new array `scores` with the same size as the bins.
4.  Iterate through the cumulative capacities and calculate the priority scores for each bin.
5.  The priority score for each bin is calculated as the negative ratio of its capacity to its current cumulative capacity.

This approach provides a more accurate representation of the priority scores, taking into account the decreasing capacity of each bin and the increasing size of the item.
