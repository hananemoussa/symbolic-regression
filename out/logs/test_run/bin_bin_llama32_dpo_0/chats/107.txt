```python
import numpy as np

def priority(item: float, bins: np.ndarray) -> np.ndarray:
    """
    Returns priority with which we want to add item to each bin.

    Args:
        item: Size of item to be added to the bin.
        bins: Array of capacities for each bin.

    Return:
        Array of same size as bins with priority score of each bin.
    """
    
    # Sort bins in descending order of their capacities
    # This is done to place the item with the largest capacity first
    # which reduces the number of unnecessary comparisons in the final sorting step
    bins = np.sort(bins)[::-1]
    
    # Initialize an array to store the priority scores
    scores = np.zeros_like(bins)
    
    # Iterate over the bins in reverse order (from largest capacity to smallest)
    for i in range(len(bins) - 1, -1, -1):
        # Calculate the priority score for the current bin
        # It is the negative of the difference between the current bin's capacity and the item's size
        # This is because the item should be placed in the bin with the largest capacity first
        scores[i] = - (bins[i] - item)
    
    return scores
```

2.  **Sorting Backward**: The function then iterates over the bins in reverse order (from largest capacity to smallest) instead of in ascending order. This is done by using the `range` function with `len(bins) - 1, -1, -1` as the loop variable.
3.  **Negative Priority Scores**: The priority scores are calculated as the negative of the difference between the current bin's capacity and the item's size. This ensures that the item is placed in the bin with the largest capacity first, which is the desired behavior.
