```python
import numpy as np

def priority(item: float, bins: np.ndarray) -> np.ndarray:
    """
    Returns priority with which we want to add item to each bin.

    Args:
        item: Size of item to be added to the bin.
        bins: Array of capacities for each bin.

    Return:
        Array of same size as bins with priority score of each bin.
    """
    # Calculate the effective capacity of each bin
    # considering that not all bins can be filled completely
    effective_bin_capacities = bins + item * (bins < item)

    # Sort the bins in descending order of their effective capacities
    sorted_bins = np.argsort(effective_bin_capacities)[::-1]

    # Assign the item to the bin with the highest effective capacity
    bin_indices = np.concatenate((range(len(bins)), sorted_bins))
    priority = -effective_bin_capacities[bin_indices]
    return priority
```

**Why it works**
----------------

1.  **Efficient binning**: By considering the effective capacity of each bin (i.e., the maximum item size that the bin can hold), we avoid unnecessary reassignments.
2.  **Optimized sorting**: Sorting the bins in descending order of their effective capacities ensures that the most efficient bin is assigned the item first.
3.  **Reduced reassignments**: By assigning the item to the highest-capacity bin, we minimize reassignments, reducing the overall number of items added to bins.

*   It reduces the number of items added to bins by minimizing reassignments, resulting in a lower total score.
