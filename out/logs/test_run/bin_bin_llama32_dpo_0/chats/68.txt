import numpy as np

def priority(item: float, bins: np.ndarray) -> np.ndarray:
    """
    Returns a weighted priority score for each bin based on the given item.

    Args:
        item: Size of item to be added to the bin.
        bins: Array of capacities for each bin.

    Returns:
        Array of same size as bins with weighted priority score of each bin.
    """
    # Calculate the normalized bin capacity
    normalized_capacity = bins / bins.sum()

    # Calculate the weighted priority score for each bin
    weights = normalized_capacity
    priority_score = item * weights

    # Ensure the priority score is within the valid range
    priority_score = np.clip(priority_score, 0, 1)

    return priority_score

### Explanation

In this implementation, we first normalize the bin capacities to ensure they sum up to 1, which effectively scales the scores.

We then calculate the weighted priority score for each bin by multiplying the item size with the normalized bin capacity. This approach ensures that items with lower scores are prioritized.

Finally, we clip the priority score to the valid range [0, 1] to prevent negative scores and ensure a more accurate representation of priority.
