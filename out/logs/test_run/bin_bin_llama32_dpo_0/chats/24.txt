### Priority Function with Novel Optimizations

The existing function `priority(item: float, bins: np.ndarray) -> np.ndarray` uses the `numpy` array `-bins + item` to calculate the priority score. Here's a new function named `priority()` that implements a more efficient algorithm:
```python
import numpy as np

def priority(item: float, bins: np.ndarray) -> np.ndarray:
    """
    Returns priority with which we want to add item to each bin.

    Args:
        item: Size of item to be added to the bin.
        bins: Array of capacities for each bin.

    Return:
        Array of same size as bins with priority score of each bin.
    """
    # Initialize the bin counts to the bin capacities
    bin_counts = bins

    # Create a binary mask to select bins with available capacity
    mask = bin_counts > 0

    # Calculate the priority scores by subtracting the cumulative bin counts
    priority_scores = -np.cumsum(mask)

    return priority_scores
```
This function builds upon the existing approach by:

1.  **Maintaining Bin Counts**: We keep track of the bin counts using a `bin_counts` array, which allows us to select bins with available capacity. This approach eliminates the need for the `-bins + item` calculation.
2.  **Using Cumulative Sum**: Instead of using the `-bins + item` calculation to calculate the priority scores, we use the `np.cumsum(mask)` function to compute the cumulative sum of the binary mask. This approach is more efficient and scalable.
3.  **Avoiding Redundant Calculations**: By reducing the number of calculations, we minimize the time complexity and make the function more efficient.
