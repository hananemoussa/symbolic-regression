```python
import numpy as np

def priority(item: float, bins: np.ndarray) -> np.ndarray:
    """
    Returns priority with which we want to add item to each bin.

    Args:
        item: Size of item to be added to the bin.
        bins: Array of capacities for each bin.

    Return:
        Array of same size as bins with priority score of each bin.
    """
    # Sort bins in ascending order based on their available capacity
    bins = np.sort(bins)[::-1]
    
    # Initialize an array to store the priority score for each bin
    priority_scores = np.zeros_like(bins)
    
    # Iterate over each bin and calculate its priority score
    for i, bin in enumerate(bins):
        # Calculate the distance of the item from the current bin
        distance = np.abs(item - bin)
        
        # Assign a priority score to the current bin based on its distance from the item
        # Assign higher priority scores to bins that are closer to the item
        priority_scores[i] = -distance
    
    return priority_scores
```

**What's happening here?**

*   We first sort the bins in ascending order based on their available capacity. This is a key insight that helps us to assign higher priority scores to bins that are closer to the item.
*   We then initialize an array to store the priority scores for each bin.
*   We iterate over each bin and calculate its priority score based on its distance from the item. We assign a higher priority score to bins that are closer to the item.
*   Finally, we return the array of priority scores for each bin.

**Why this approach?**

*   By sorting the bins based on their available capacity, we ensure that bins with the most available capacity are assigned the highest priority scores.
*   This approach also helps to avoid assigning a negative priority score to bins that are not available, which would have resulted in an incorrect assignment of priority scores.
